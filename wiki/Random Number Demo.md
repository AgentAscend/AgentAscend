

  

## Summary

Example application demonstrating token-gated access using Solana payments.

  

## Flow

1. User connects wallet

2. User pays 0.1 SOL

3. Backend verifies payment

4. User can generate random number (0–1000)

  

## Components

- Frontend (Next.js)

- Payment SDK

- Backend verification

- Hermes agent logic
