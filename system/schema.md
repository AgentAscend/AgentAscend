# Wiki Schema

  

## Purpose

Defines how all wiki pages should be structured.

  

## Standard Format

  

Each page should include:

  

## Summary

Short explanation of the concept

  

## Components

List of important parts

  

## Relationships

Links to other pages using [[wikilinks]]

  

## Notes

Extra important details

  

---

  

## Naming Rules

- Use clear names (e.g. "Payment System", not "payments_v2")

- One concept per page

- Avoid duplication

  

---

  

## Linking Rules

- Always link related pages

- Use [[Page Name]] format

- Build a connected graph, not isolated notes

  

---

  

## System Rules

- Do NOT place raw notes in /wiki

- Do NOT place structured pages in /raw

- Keep /system reserved for rules and purpose only
